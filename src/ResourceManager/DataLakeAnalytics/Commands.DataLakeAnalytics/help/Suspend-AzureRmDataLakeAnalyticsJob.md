# Suspend-AzureRmDataLakeAnalyticsJob

## SYNOPSIS
Yields a job.

## SYNTAX

```
Suspend-AzureRmDataLakeAnalyticsJob [-Account] <String> [-JobId] <Guid> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
```

## DESCRIPTION
The **Suspend-AzureRmDataLakeAnalyticsJob** cmdlet yields an Azure Data Lake Analytics job.

## EXAMPLES

### Example 1
```
PS C:\>Suspend-AzureRmDataLakeAnalyticsJob -Account "ContosoAdlAccout" -JobId "a0a78d72-3fa8-4564-9b18-6becb3fda48a"
```

This command yields the job with the specified ID.
